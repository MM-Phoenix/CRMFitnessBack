ля запуску CRMFitnesBack проекту потрібно завантажити java не менше 17 версії.  Далі потрібно завантажити і встановити PostgreSQL, створити базу даних “crm_fitness” та створити користувача “postgres” із паролем “hakkAc-jesky1-zuvhoj”.  Далі потрібно завантажити з репозиторія github “Backend” з додаткуВ jar файл “CRMFitnessBack-1.0.0-SNAPSHOT.jar” і запустити консоль в папці із jar файлом і виконати команду “java -jar CRMFitnessBack-1.0.0-SNAPSHOT.jar”.
